# Cloned Repositories

## Repo 1: Toy Models of Superposition
- URL: https://github.com/anthropics/toy-models-of-superposition
- Purpose: Synthetic experiments on superposition and interference.
- Location: code/toy-models/
- Key files: `toy_models.ipynb`
- Notes: Used to study phase changes in feature representation.

## Repo 2: Geometry of Truth
- URL: https://github.com/saprmarks/geometry-of-truth
- Purpose: Probing and intervening on semantic concepts like truth.
- Location: code/geometry-of-truth/
- Key files: `generate_acts.py`, `probes.py`, `interventions.py`
- Notes: Contains many useful datasets in `datasets/`.

## Repo 3: TransformerLens
- URL: https://github.com/TransformerLensOrg/TransformerLens
- Purpose: Mechanistic interpretability of transformers.
- Location: code/transformer-lens/
- Notes: Standard library for extracting activations and patching models.

## Repo 4: SAELens
- URL: https://github.com/jbloomAus/SAELens
- Purpose: Training and analyzing Sparse Autoencoders.
- Location: code/saelens/
- Notes: Useful for decomposing representations into interpretable features.

## Repo 5: Schrödinger's Bat
- URL: https://github.com/rycolab/diffusion-polysemy
- Purpose: Polysemy and superposition in diffusion models.
- Location: code/schrodingers-bat/
- Key files: `experiments.ipynb`
- Notes: Demonstrates visual superposition via weighted CLIP encodings.
